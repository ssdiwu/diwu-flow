# .doc/ 导航索引

> 本目录是 diwu-flow 的**产品文档层**：导航、边界定义、传播矩阵。
> 不复读 `README.md`（用户视角）或 `.claude/CLAUDE.md`（Agent 行为规范）的内容。

## 文档定位

| 文件 | 回答什么 | 不回答什么 |
|------|---------|-----------|
| [架构规范.md](./架构规范.md) | 模块如何协作、Hooks 链路、数据流向 | 具体 Skill 步骤、Command 用法 |
| [状态文件规格.md](./状态文件规格.md) | dtask.json / dtask-state.json 字段语义与写者矩阵 | 规则文件内容、Skill 方法论 |
| [工程规范.md](./工程规范.md) | 变更传播规则、禁止事项、引用索引 | 具体实现细节 |

## 推荐阅读顺序

1. **架构规范** — 理解整体骨架和事件驱动链
2. **状态文件规格** — 理解数据真相源和写者责任
3. **工程规范** — 理解修改时的约束和传播路径

## 维护规则

- 本目录不维护静态 inventory（如 Command/Skill 计数），计数以 `plugin.json` 为准
- 新增文档时更新本索引表
